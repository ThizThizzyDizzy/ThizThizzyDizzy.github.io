NuclearCraft Plannerator

A tool for designing NuclearCraft multiblocks
This comes in two forms:
The Standard Edition, for windows/mac/linux desktop, which can be found on GitHub.
The Total World Domination Edition, for android, which can be found on the Google Play Store. (and also GitHub)
Note that despite the fancier name, the android version is much more watered down, and does not provide many of the features present in the standard edition.
